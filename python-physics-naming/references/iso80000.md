# ISO/IEC 80000 Symbol Tables — Python Transliterations

These are the Python-safe transliterations of ISO/IEC 80000 quantity symbols. The standard itself prints them in italics; in Python you use plain Latin letters (Greek letters spelled out). Use these as the default when a paper or textbook notation does not dictate otherwise.

Each table lists: **symbol (Python)** — quantity — SI unit — notes.

## When two quantities share a symbol

This happens constantly (`E` = energy *and* electric field; `T` = period *and* temperature; `n` = number density *and* refractive index *and* amount of substance). Resolution:

1. **Let module scope disambiguate.** A thermodynamics module: `T` means temperature. An optics module: `n` means refractive index. Document this at module level.
2. **If both appear in the same scope**, disambiguate with a subscript: `T_period`, `T_temp`, `n_ref`, `n_density`.
3. **Never silently overload**. The reader should never have to guess which `E` is in play.
